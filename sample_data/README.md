# Sample Data

This directory contains sample datasets for practicing data analysis and visualization.

## Datasets

### employees.csv
A CSV file containing employee information with the following columns:
- **Name**: Employee name
- **Age**: Employee age
- **Department**: Department (Engineering, Marketing, Sales)
- **Salary**: Annual salary in dollars
- **Years_Experience**: Years of work experience

**Use cases:**
- Practice reading CSV files with pandas
- Group by operations and aggregations
- Statistical analysis
- Data visualization (bar charts, scatter plots, box plots)

### products.json
A JSON file containing product catalog information with the following structure:
- **id**: Product ID
- **name**: Product name
- **category**: Product category
- **price**: Product price in dollars
- **stock**: Available inventory
- **ratings**: List of customer ratings (0-5 scale)

**Use cases:**
- Practice reading JSON files
- Working with nested data structures
- Calculating statistics (average ratings)
- Inventory analysis

## Adding Your Own Data

Feel free to add your own datasets to this directory for practice. Common formats include:
- CSV files (.csv)
- JSON files (.json)
- Excel files (.xlsx)
- Text files (.txt)

Remember to update this README if you add new datasets!
