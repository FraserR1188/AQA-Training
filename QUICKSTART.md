# Quick Start Guide

Welcome to the AQA Training repository! This guide will help you get started quickly.

## Initial Setup

1. **Install Python dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Launch Jupyter:**
   ```bash
   # For Jupyter Notebook (classic interface)
   jupyter notebook
   
   # OR for JupyterLab (modern interface)
   jupyter lab
   ```

3. **Open your first notebook:**
   - Navigate to `notebooks/basics/01_jupyter_basics.ipynb`
   - This will introduce you to Jupyter notebook fundamentals

## Learning Path

Follow these notebooks in order:

### 1. Basics (Start Here)
- `notebooks/basics/01_jupyter_basics.ipynb`
  - Learn Jupyter notebook interface
  - Basic Python syntax
  - Variables, functions, and data structures

### 2. Data Analysis
- `notebooks/data_analysis/01_pandas_basics.ipynb`
  - Loading CSV and JSON data
  - Data manipulation with pandas
  - Basic statistical analysis
  - Filtering and grouping data

### 3. Visualization
- `notebooks/visualization/01_matplotlib_seaborn.ipynb`
  - Creating charts with matplotlib
  - Advanced visualizations with seaborn
  - Bar charts, scatter plots, histograms, box plots
  - Multi-panel visualizations

## Sample Data

The `sample_data/` directory contains practice datasets:
- **employees.csv**: Employee data for analysis practice
- **products.json**: Product catalog in JSON format

See `sample_data/README.md` for details about each dataset.

## Tips

- **Save frequently**: Use `Ctrl+S` (or `Cmd+S` on Mac) to save your work
- **Run cells**: Use `Shift+Enter` to run a cell and move to the next one
- **Restart kernel**: If something goes wrong, try `Kernel > Restart & Clear Output`
- **Add cells**: Use the `+` button in the toolbar to add new cells
- **Cell types**: Use the dropdown to switch between Code and Markdown cells

## Next Steps

After completing the starter notebooks:
1. Experiment with the sample data
2. Add your own datasets to `sample_data/`
3. Create new notebooks for your experiments
4. Apply techniques to your main project
